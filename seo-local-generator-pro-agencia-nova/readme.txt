Plugin profissional para geração de páginas geolocalizadas otimizadas para SEO com suporte a variáveis dinâmicas e layouts personalizados.

== Descrição ==

Este plugin permite que você gere centenas de páginas automaticamente combinando listas de produtos e cidades, utilizando templates personalizados com estrutura de SEO local.

Recursos principais:

- Geração de páginas estáticas com slug otimizado
- Substituição de variáveis [produto], [cidade], [empresa], [formulario], [whatsapp]
- Botão de WhatsApp dinâmico
- Suporte a formulário do Contact Form 7 por produto
- Layouts para clínicas, indústrias, escritórios e modelo padrão
- Totalmente compatível com Yoast SEO

== Instalação ==

1. Envie o plugin para o diretório `/wp-content/plugins/`
2. Ative o plugin através do menu "Plugins" no WordPress
3. Acesse "SEO Local Generator" no menu do painel para configurar
